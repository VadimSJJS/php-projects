Описание проекта "Ежедневная сказка"

Ссылка на проект

Проект представляет собой сайт, ежедневно выводящий новую сказку для детей.

Описание

На главной странице сайта должна выводиться одна новая сказка для детей. Каждый день история сказка должна меняться на новую.

Возраст

Сайт должен выдавать сразу несколько сказок на различный возраст: от 0 до 3, от 3 до 7, от 7 до 12. Пользователь сайта может регулировать с помощью переключателя, сказку из какой группы он хочет увидеть. Сайт должен запоминать выбор пользователя и в следующий раз по умолчанию показывать сказку из выбранной им группы.

Режимы

Сайт должен поддерживать дневной и ночной режим. В ночном режиме сайт должен иметь темный фон.

Админка

Истории должны добавляться через админку. Админ должен иметь возможность добавлять истории на любое количество дней вперед.
